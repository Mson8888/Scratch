Tetris — Starter
=================

This repository contains a minimal starter for building a Tetris game that can be hosted on GitHub Pages.

Getting started
---------------

1. Edit the files in this repo (index.html, styles.css, script.js) to implement the game.
2. Commit your changes: git add . && git commit -m "feat: implement Tetris" -m "Co-authored-by: <Your Name> <you@example.com>"
3. Push to GitHub: git remote add origin <repo-url> && git push -u origin main

Publishing on GitHub Pages
-------------------------
- Go to your repository Settings → Pages and choose Branch: main / folder: / (root), then Save. GitHub will publish the site at https://<username>.github.io/<repo-name>/.

Notes
-----
- This is a starter scaffold. The JavaScript contains a simple game loop and rendering scaffold to help you begin.
- If you want automatic deployments, consider adding a GitHub Action for Pages or using the gh-pages npm package.
